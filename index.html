Welcome to NimbleX Linux!

NimbleX is a small but versatile operating system which is able to boot from CD, flash memory (USB pens or Mp3 players) and even from the network. Because it runs entirely from a CD, USB or network it doesnâ€™t require installation or even much hardware.

Features

NimbleX was designed to be versatile, user friendly, very fast and most importantly indestructible.

Usage

Currently NimbleX is provided mostly as the collection of scripts to build the distro yourself but you can also download the latest ISO image from here.

Contributors

The project definitely can benefit from contributions that want to create the next fast and user friendly OS. If you can write documentation or code get in touch and we'll be glad to have you in our team.
